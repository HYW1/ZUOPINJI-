# AI Portfolio 作品集

一个基于设计稿方向搭建的响应式单页作品集，可用于展示个人介绍、精选项目、工作流程、能力关键词与联系方式。

> 说明：当前环境无法连接 Figma MCP，也无法直接抓取该 Figma 文件内容，因此本版本先交付可运行、可替换内容的作品集页面。拿到设计稿截图、导出的图片或恢复 Figma 访问后，可以继续逐项对齐颜色、间距、字体和素材。

## 文件结构

```text
.
├── index.html   # 页面结构与内容
├── styles.css   # 视觉样式、响应式布局
└── script.js    # 筛选、滚动导航状态、年份
```

## 本地预览

可以用任意静态服务器预览，例如：

```bash
python3 -m http.server 4173
```

然后打开 <http://localhost:4173>。

## 如何替换成自己的作品集

1. 在 `index.html` 中替换姓名、职位、项目标题、项目描述和联系方式。
2. 将 `.work-card` 中的示例项目替换为真实作品。
3. 如有项目封面图，可把 `.work-visual` 改为 `<img>` 或在 CSS 中替换为背景图。
4. 在 `styles.css` 的 `:root` 中调整颜色变量，快速贴近设计稿视觉。
